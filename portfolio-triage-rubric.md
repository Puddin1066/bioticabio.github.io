# Portfolio triage rubric

A practical scoring rubric for ranking programs when resources are constrained and strategic focus is required.
{: .lead }

## Scoring dimensions

- **Evidence conviction** (association quality, source breadth, reproducibility).
- **Execution feasibility** (modality fit, trial complexity, timeline realism).
- **Commercial edge** (differentiation, competitive pressure, narrative clarity).
- **Strategic fit** (portfolio coherence, capability alignment, opportunity cost).

## Suggested output categories

- **Tier 1:** resource now
- **Tier 2:** monitor and de-risk
- **Tier 3:** deprioritise or partner

## Operating guidance

- Re-score on a fixed cadence as evidence updates.
- Track score changes with rationale, not just rank changes.
- Keep one decision owner per program to avoid drift.
